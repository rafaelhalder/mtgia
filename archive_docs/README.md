# Arquivo de Documentos (archive_docs)

Esta pasta guarda documentos `.md` que não são prioridade operacional no momento.

Objetivo:
- reduzir ruído na raiz do projeto,
- manter histórico sem perda,
- facilitar foco no roadmap atual.

Se algum documento voltar a ser necessário, basta movê-lo de volta para a raiz ou para a pasta apropriada.
